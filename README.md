# NexusCore - Premium Business React Template

A high-end, modern, and highly responsive business template built with React. Perfect for agencies, startups, corporate businesses, and SaaS landing pages. 

## 🌟 Key Features

- **Premium Design System**: Modern dark aesthetics with glassmorphism `backdrop-filter` effects.
- **Fully Responsive**: Flawless layout across desktops, tablets, and mobile devices using CSS Grid & Flexbox.
- **Dynamic Animations**: Subtle floating effects, smooth hover micro-interactions, and gradient text masking.
- **Premium Typography**: Uses **Outfit** for bold, stylized headings and **Inter** for clean, readable body text.
- **High-Quality Icons**: Integrated with `react-icons` and `lucide-react` for crisp, scalable vector icons.
- **Clean Code Architecture**: Built with React functional components, highly modular, and extremely easy to customize.

## 🚀 Getting Started

Follow these steps to get the template running on your local machine.

### Prerequisites

You need to have Node.js installed. If you don't have it, download and install it from [nodejs.org](https://nodejs.org/).

### Installation

1. Open your terminal and navigate to the template directory.
2. Install all the required dependencies by running:
   ```bash
   npm install
   ```
   *(Note: This project uses `react-icons` and `lucide-react`. `npm install` ensures they are locally available).*

### Running the Development Server

Start the local development server:
```bash
npm start
```
The application will automatically open in your default browser at `http://localhost:3000`.

### Building for Production

When you are ready to deploy your site, run:
```bash
npm run build
```
This builds the app for production to the `build` folder. It correctly bundles React in production mode and optimizes the build for the best performance.

## 🎨 Customization

- **Colors & Branding**: All main colors (glows, accents, backgrounds) are defined as CSS Variables in `src/index.css`. Simply change the hex codes under `:root` to match your brand!
- **Content**: Update texts and arrays inside `src/components/*.js` files (like `Hero.js`, `Services.js`, `Contact.js`) to personalize the site for your business.
